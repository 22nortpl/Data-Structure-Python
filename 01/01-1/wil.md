알고리즘 : 컴퓨터가 문제를 해결하거나 작업 처리를 위해 따라야 하는 명확한 절차, 규칙, 명령들의 집합
  순차 구조 : 한 문장씩 순서대로 처리되는 구조
  선택 구조 : 조건식으로 평가한 결과에 따라 프로그램의 실행 흐름이 변경되는 구조

순서도 : 알고리즘을 그림으로 표현
  데이터 : 평행사변형 모양, 기억 장치를 지정하지 않은 데이터 자체

  처리 : 사각형 모양, 정보의 값, 형, 위치를 바꾸도록 정의한 연산이나 연산 집합의 샐행

  미리 정의한 처리 : 사각형에 두 줄, 서브루틴이나 모듈 등 다른 곳에서 이미 정의한 하나 이상의 연산 또는 명령으로 이루어진 처리

  판단 : 다이아몬드 모양, 하나의 입구와 하나 이상의 출구 존재, 조건 평가로 하나의 출구를 선택하는 구조

  루프 범위 : 두 부분으로 구성되어 루프의 시작과 종료를 표현

  선 : 제어의 흐름

  단말 : 시작과 종료 표현
